总述：隐式限制

游戏的本质是学习，学习的一种对象是设定，设定包括【内容】与【应用】。
如果以设定的内容来设计游戏的学习对象，那么设定的内容就不能太简单，否则很快学透了、就无聊了。
但复杂的设定也会劝退玩家。
怎么办？
答案：在简单设定的基础上做出隐式限制。

隐式限制，例如，对于LOL之类的RPG战斗游戏，装备可能会有众多词条。
简单的设定，就是【一个装备会有一些词条】。
隐式限制，就例如【伴生：如果一个装备没有词条A，那么一定不会有词条B】【互斥：如果一个装备有词条X，那么一定不会有词条Y】。
比如LOL中，【攻速】与【护甲】就是互斥的，【暴击】与【攻击】就是伴生的。

隐式限制可以让玩家既觉得设定简单、又学不透。
Opus Magnum是一个缺乏隐式限制的游戏。在这个游戏中，玩家需要组装出一个个指定结构的分子，分子结构是几乎随机的。如果能加入隐式限制，那么销量会更高。给它设计的隐式限制的结构就可以是【满足某条件的分子不会存在】，比如现实中【化学键不正确的分子不会存在】。
*******************************
概念

R设定：游戏中的一种【存在大量的一阶下级】的设定。例如LOL中的装备、Opus中的分子、三国杀中的武将。
理论R空间：例如任意词条组合的结果都属于LOL装备理论空间，任意可描述的武将都属于三国杀武将理论空间。
约束R空间：考虑到平衡性以及其他原因，从理论R空间中剔除掉不该存在的设定，剩余的就是约束R空间。
真实R空间：游戏中实际的所有R设定的一阶下级。比如LOL中实际的所有装备。
隐式限制：形如【真实R空间中不会存在满足XX条件的元素】的规则。理论R空间+隐式限制=约束R空间。

之所以叫隐式限制，是因为游戏规则中并不会说【不会存在满足XX条件的R设定】。这个规则是隐藏的。
玩家看到的是真实R空间，游戏规则说的是理论R空间。玩家试图以理论R空间来认知这个游戏，但是会屡屡失败：这是因为需要用约束R空间来认知游戏。
如果一个游戏的理论R空间等于约束R空间（即没有隐式限制），那么会很快被玩家看透、无聊。

隐式限制·包

《三国杀》《游戏王》中有【包】的概念。
在三国杀中，一个【包】包含多个武将。每个包都会引入几个新的效果：只有这个包内的武将才会有这个效果（一般来说）。
在游戏王中，一个【包】包含多个卡牌。每个包会引入几个新的关键词，包内的卡牌的效果会基于这些关键词来联动。
在RPG游戏中，【包】表现为【地图】，不同地图掉落的装备的风格不同，单个地图掉落的装备会有一些共性。（但有些RPG游戏中地图不完全是包，我认为这是作者的设计不好。）

包还可以促进【数值限制】。可以设计这样的隐式限制：某个包中【当一件装备同时存在A词条与B词条时，A的数值一定大于B的四倍】，在另一个包中【当一件装备同时存在A词条与B词条时，A的数值一定小于B】。

隐式限制·次元交汇

《植物大战僵尸》中有多个行，各个行之间原本独立运行。但后期会有【樱桃炸弹】【玉米炮】【西瓜】【忧郁蘑菇】之类的跨行效果。这就是次元交汇。多个行是多个次元，大多数设定在次元内部，少量设定跨次元运行。
没有这样的隐式限制的游戏，则是【所有防御塔都是跨行的】，那样会无聊。
《EVE》中，舰船分为四类，装备也分为四类，一类舰船只能使用对应的一类装备。但还有少量的特殊装备，允许所有舰船使用。例如，在EVE中，低级舰船使用低级炮台，伤害低；高级舰船是高级炮台，伤害高。但是无论什么舰船，都可以使用【伤害增幅器】，它会百分比提升伤害。
EVE也属于比较简陋的次元交汇。

在我的游戏中，以五行划分次元。五行是五个数字，角色与装备都有五行。只有当角色五行的每一项都大于等于装备五行时，才可以穿戴此装备。
前期有这样三个次元：【100 1 1 1 1】【95 2 2 2 2】【90 3 3 3 3】。三类角色，三类装备，每一类角色只能使用自己类的装备。
但是会有少量特殊角色，五行例如【95 3 3 3 3】，这样的角色就可以穿戴后两类装备，这就属于次元交汇。

我的游戏中会有十多个次元，每个次元相当于一个独立的游戏；每个次元是一个包，有专属的设定，有完整的玩法（专属角色、装备、地图）。后期，这些次元会交汇，那也是真正游戏的开始（前期也会少量交汇）。
